A Flask-based web application for logging workouts with the ability to print custom Word templates and save data by session or user login. Perfect for tracking gym progress and generating printable workout summaries.

Project Overview

This workout logging application was born out of frustration with existing fitness apps that were either too complex or didn't allow for the specific tracking format I wanted. The app allows users to quickly log exercises, sets, reps, and weights while providing the ability to generate formatted printouts for offline reference.

Key Features

  • Quick workout entry with exercise autocomplete
  • Session-based tracking with optional user accounts
  • Custom Word document generation for printable logs
  • Exercise database with common gym exercises
  • Progress tracking and historical data
  • Responsive design for mobile and desktop use

Technical Implementation

Built with Flask and SQLite for lightweight deployment, the app uses Python-docx for Word document generation and Flask-Login for user management. The frontend uses Bootstrap for responsive design and jQuery for dynamic form handling.

The Word template feature was particularly interesting to implement - it dynamically populates workout data into a pre-formatted template that maintains consistent formatting across different workout types.

User Experience Features

  • Quick Entry: Streamlined form for rapid workout logging
  • Smart Autocomplete: Exercise names populate as you type
  • Session Management: Group exercises by workout session
  • Mobile Friendly: Optimized for gym floor use on phones

Data Management

  • SQLite Database: Lightweight, no server required
  • User Accounts: Optional login for personal data
  • Export Options: Generate Word docs or CSV files
  • Data Backup: Easy database backup and restore

Development Process & Challenges

The biggest challenge was balancing simplicity with functionality. I wanted something faster than a notebook but more flexible than existing apps. The Word template generation took several iterations to get right - ensuring consistent formatting while handling variable amounts of data.

Originally built for personal use, I've since shared it with friends who appreciate having their own local instance rather than relying on cloud-based fitness apps. The next version will include exercise instruction videos and more detailed analytics.
