# Toms in the lab - Portfolio Website

A personal portfolio website showcasing maker projects and technology solutions by Tom Battista.

## Project Overview

This website features:
- Personal portfolio with project showcase
- Responsive design using Bootstrap
- Individual project detail pages
- Contact form and social media integration

## Structure

- `index.html` - Main homepage
- `portfolio-masonry.html` - Projects grid view
- `contact.html` - Contact page
- `projects/` - Individual project pages
- `docs/` - Project documentation
- `css/`, `js/`, `images/` - Assets
- `includes/` - Shared components

## Projects Featured

1. **Gym Workout Logger** - Flask-based web application for fitness tracking
2. **Wiffle Ball Club Website** - Community website for Wizards Wiffle Ball Club
3. **Spicy - Spice Inventory App** - Mobile app for kitchen spice management
4. **PARA Notion Dashboard** - Productivity system using the PARA method
